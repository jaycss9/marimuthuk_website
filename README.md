## 🚀 Quick Start

### Option 1: Direct Open
Simply open `index.html` in your web browser.

### Option 2: Local Server
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000`

### Option 3: Live Server (VS Code)
1. Install "Live Server" extension
2. Right-click on `index.html`
3. Select "Open with Live Server"

## 🎭 Customization

### Changing Colors
Edit CSS custom properties in `modern-style.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #06b6d4;
    ...
}
```

### Updating Content
All content is in `index.html` with semantic HTML structure for easy editing.

### Adding New Sections
Follow the existing section pattern with proper classes for automatic animations.
